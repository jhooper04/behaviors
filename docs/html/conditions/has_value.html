Condition has_value

The has-value node inherits from the condition base node class and will check if the associated node's set object has a property defined (checks if a key is nil on the object).

Functions

constructor (config) Constructs a has_value node class instance.
check_condition (...) The main method that handles the check for this has_value node's object key is not nil.
