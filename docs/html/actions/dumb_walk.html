Action dumb_walk

The dumb_walk node inherits from the action base node class and applies a velocity toward a target position and setting a walking animation.

The action will either succeed on timeout or when the mob reaches the target position.

on_step (...)
The main method that handles the processing for this dumb_walk node. This node will cause the mob to walk towards the set target position vector without regard to any obstacles or height changes. returns success when the mob reaches the target position or the operation times out.

Parameters:

  • ... Any parameters passed to the node's run method

Returns:

    A string representing the enum behaviors.states of "running", or "success".

Tables

config
Configuration table passed into the constructor function.

Fields:

  • walk_animation string The walking animation to apply to the mob while walking.
  • speed_factor number How fast the mob should walk.
  • timeout number How long should this node should attempt to walk toward the position before it gives up.
object
Required properties set on the node's set object

Fields:

  • target_pos vector The position vector to walk to in the form {x,y,z} or an actual vector type.
