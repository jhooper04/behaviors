Base node

on_step [abstract]
The main method that handles the processing for this node. Subclasses will override this method to perform specific tasks relevant to the node's individual types. The node base class only has an empty stub method that just throws a not implemented error.

Parameters:

Returns:

    A string representing the enum behaviors.states of "running", "success", or "failed".
